# Folio — Personal Book Library

Folio is a fully functional, elegant single-page personal book library web app built using the MERN stack (MongoDB, Express.js, React, Node.js). It provides a premium digital bookshelf experience with a warm, sophisticated aesthetic inspired by modern productivity tools and physical library catalogs.

## Features
- **Book Management:** Add, view, edit, and manage your personal book collection seamlessly.
- **Search & Filtering:** Effortlessly find books using advanced search and category/genre filters.
- **Activity Tracking:** Keep track of borrowing history, reading progress, and current book status.
- **Elegant UI:** A responsive, eye-catching interface featuring a bespoke 3D book spine effect, smooth micro-animations, and a curated typography-driven design.
- **Robust Backend:** RESTful API connected to a MongoDB database for reliable data persistence.

## Tech Stack
- **Frontend:** React (Vite), React Router, Vanilla CSS (Design Tokens & Variables)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Tooling:** Git, npm
